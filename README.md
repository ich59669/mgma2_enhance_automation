# mgma2_enhance_automation

## このスケッチについて
Nintendo Switch版の「魔物娘と不思議な冒険2」で鍛冶、闘技場、果実畑の自動化するためのArduino Leonardo用スケッチです。

本マクロで生じたゲーム内外の損失に於いて、制作者は責任を負いません。自己責任でご利用ください。

Switchの操作に interimadd 氏の[NintendoSwitchControl](
https://github.com/interimadd/NintendoSwitchControll)を利用しています。\
上記ライブラリをインストールしてください。\
Arduinoの導入についても氏のリポジトリが詳しいので、初めての方はこちらを読むことをおすすめします。

準備したハードウェアに応じた3種類のスケッチを用意しています。\
Arduinoのみで動作する場合は、手動開始や一時停止、及び周回数設定はできません。\
ボタンを実装した場合は、手動開始や一時停止ができます。周回数設定はできますが、ボタンを押した回数を自分で数える必要があります。\
7セグLEDを実装した場合は、状態や残り周回数を確認できるようになります。

## 自動周回実行中に低確率で発生する事象
- 闘技場1Fのロードで時間が異常にかかり、その後の操作がずれてしまう(原因究明中)\
以降まともに動かなくなります。手動で復帰してください。
- 階段に向かう途中に、ドロップした**果実**があった場合\
育てたい果実より上に来てしまうと、ドロップした果実が優先して栽培されてしまいます。
- 仲魔が目の前にいて罠チェックができず、その場所に致命的な罠があった場合\
特にアイテム落としの罠だった場合、育てている装備や栽培した果実をロストするおそれがあります。\
**画面を表示しながら別作業をするような運用が望ましいと思われます。**

## スケッチの変更
それぞれのスケッチフォルダの`data.h`を編集します。
- mgma2_auto_run.ino\
`const uint32_t START_DELAY_MS = 10000; // 起動から開始までの待ち時間 (ms)`\
こちらの値を環境に合わせて変更してください。デフォルト値は10000ms(10秒)です。

- mgma2_auto_button.ino\
`const byte buttonPins[2] = { 8, 9 };`\
こちらの値をボタンを接続したピン番号にあわせて変更してください。左が設定ボタン、右が開始ボタンです。
`const int colosseumLoopTimeSecond = 435;`\
鍛冶屋と闘技場を１周するのにかかる時間(秒)です。\
`const int fruitRipenTimeSecond = 8830;`\
果実が実るまでの時間(秒)です。

- mgma2_auto_sevseg_button.ino\
```
const byte digitPins[4] = { 10, 11, 12, 13 };        // 1桁目から昇順
const byte segLedPins[7] = { 7, 5, 3, 1, 0, 6, 4 };  // セグメントAから
const byte dpPin = 2;
```
7セグLEDの配線に合わせて、変更してください。

## スケッチ書き込み後の使用方法
1. ゲーム側をあらかじめセットアップし、先に使用したコントローラの接続を切っておきます。
2. ArduinoをUSBでSwitchと接続します。
3. Arduino単体動作の場合は設定時間後に鍛治の強化からスタートします。それ以外の場合は開始ボタンを押すとスタートします。

## 初回の周回数設定について
果実が実るまでの残り時間に合わせて初回の周回数のみ変更することができます。\
初期値は「果実が実るまでの時間を超える最小の周回数」で、\
設定ボタンを1度押すと「1周」に戻り、以降押すごとに1周ずつ増えます。\
7セグLEDには、周回にかかる時間が`H.MM.S`(Sは10秒単位)で表示されます。\
果実が実るまでの残り時間を超えて、かつできるだけ小さくなるように設定してください。\
ボタンのみ実装の場合でも、周回数設定の挙動は同じです。\
Arduino単体動作の場合は、周回数21固定です。(2:27:10を超える最小の周回数)

## 周回動作中の表示
数字はマクロのステップ番号です。\
闘技場と鍛治の残り周回数が、4つのDPで2進数表示されます。

## 終了したいとき
Arduinoのリセットボタンを押すか、USBを抜いてください。

## 必要なもの (ゲーム側)
### アイテム
- 「全員斬り」のスキルと「罠」(正面の罠を壊す)の印が入っていて、闘技場の敵を全員斬りで一掃できる攻撃力の剣
- 闘技場4Fのプロテクトシザーの攻撃を何度か受けても十分耐えられる防御力の盾
- 「精神」印が5つ入った魔法の腕輪
- 鍛冶屋で育てる装備
- 十分なお金

### 仲魔
- 闘技場4Fのプロテクトシザーに苦戦しない程度の強さを持つ仲魔を３体連れてください
- 仲魔への指示は「自由に行動」にしてください
- 状態異常系の特技は切っておくことが望ましいでしょう

## ゲーム側のセットアップ
1. 本編をクリアしておきます。
2. ターシャの園芸店から左右と下を塞ぐように果実畑を配置します。
3. 果実畑に育てたい果実を植えます。
4. 上記アイテムのみを持ち、必要な装備をし、「希望の村」にいるようにします。

## 周回動作
0. コントローラの接続、認識
1. 鍛治の実行
2. 闘技場の実行
3. 1と2をn回繰り返す
4. 果実畑の収穫と栽培
5. 1に戻る
